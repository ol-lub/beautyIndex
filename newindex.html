Инструкция

Порядок работы:

  1. Нажмите на кнопку "Выберите файл" и выберите фото.
  2. Расставьте точки по списку, приведенному ниже. Первые - красные, вторые - синие.
  3. Получите результат (чем ближе результат к 0, тем ваши пропорци ближе к идеальным).
  4. Средний показатель индекса - 0.4.

Порядок расположения точек:

  • Высота лица(синие) / ширина лица(красные);
  • Центральная точка соединения губ до основания носа(синие) / длина носа(красные);
  • Высота лица(синие) / расстояние от кончика подбородка до центральной точки соединения губ(красные);
  • Ширина рта(синие) / ширина носа(красные);
  • Ширина носа(синие) / расстояние между ноздрями(красные);
  • Расстояние между зрачками(синие) / расстояние между бровями(красные).
